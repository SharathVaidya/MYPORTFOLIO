## 🚀 Getting Started

To run the project locally:

```bash
# 1. Clone the repository
git clone https://github.com/SharathVaidya/MYPORTFOLIO.git

# 2. Navigate into the folder
cd MYPORTFOLIO

# 3. Install dependencies
yarn install

# 4. Start the development server
yarn start
